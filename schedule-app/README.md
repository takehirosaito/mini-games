# Schedule App

ブラウザだけで動くシンプルなスケジュール管理アプリです。

## 機能

- 月間カレンダー表示
- 予定の追加・編集・削除
- 日付、開始時刻、終了時刻、カテゴリ、メモ
- ブラウザの `localStorage` に自動保存
- 検索、カテゴリ絞り込み
- JSONエクスポート / インポート
- CSVエクスポート

## 使い方

`index.html` をブラウザで開くと利用できます。

GitHub Pages が有効な場合は、以下のURLでアクセスできます。

https://takehirosaito.github.io/mini-games/schedule-app/
